Building H2O

Sistema operacional usado: Linux
Distribuição: Ubuntu 20.04.1 LTS

Foram feitas duas implementações do PROBLEMA BUILDING H2O (seção 5.6), a primeira versão usando 3 threads e a segunda usando 2 threads. Após rodar a primeira versão por dez mil vezes, nenhum problema foi encontrado, entretando a execução da segunda implementação apresentou um falha no longo da bateria de execuções. 

Os parametros podem ser ajustados nas macros dos códigos, para realizar a compilacao basta digitar "make" no terminal. As informacoes em relacao a execucao do programa sao impressas na tela
